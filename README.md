# PokeBinds

## Sobre o Projeto:
Teste para Binds. Sistema conectado com a API PokeApi que exibe lista completa de Pokémons ou lista através de busca/filtro pré-selecionado pelo usuário.

## 🛠 Tecnologias

- VueJS
- Axios
- Bulma

## 🤔 Como executar o Projeto? 

### Requisitos

- NPM ou Yarn
- Vue

### Rodando

```bash
# Instação de dependências
$ yarn install

# Executando projeto
$ yarn serve
```
